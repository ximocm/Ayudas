# Ayudas master bioinformatica UV

Material corto y practico para alumnos del master.
Todo esta escrito para seguir pasos simples, con pocas opciones y sin liarse.

NOTA: Tengo un teclado americano, no hay enyes ni acentos sorry.

## Orden recomendado

1. `crear_entorno.md`
- Explica que es un entorno y como crearlo de forma facil.

2. `clase_bioestadistica.md`
- Crea el entorno de la asignatura desde `enviroment.yml`.

3. `ejecutar_r_desde_servidor.md`
- Complementa los anteriores para ejecutar R en servidor con VSCode.

## Archivos de este repo(de momento)

- `crear_entorno.md`: guia base de entornos
- `clase_bioestadistica.md`: entorno de la clase
- `ejecutar_r_desde_servidor.md`: ejecucion remota de R
- `enviroment.yml`: dependencias de bioestadistica
